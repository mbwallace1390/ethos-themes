# Lunar Command v1.0.0

**Collection:** Cosmic

A standalone FrSky ETHOS radio theme with custom toolbar artwork.

- Focus: `outline`
- Controls: square
- Internal key: `Lunar`
- Static 784x50 toolbar
- Does not modify Rotorflight or RF Suite Lua files

Copy `theme-lunar-command` into the transmitter `scripts` folder, restart, then select **Lunar Command** under **System > General > Theme**.
