# Soft Lavender v1.0.0

**Family:** Soft

A lightweight standalone FrSky ETHOS theme.

- Focus: `invert`
- Controls: rounded
- Internal key: `SoftLav`
- Responsive 784x50 X20 / 464x50 X18 toolbar

Copy `theme-soft-lavender` into the transmitter `scripts` folder, restart, and select **Soft Lavender**.
