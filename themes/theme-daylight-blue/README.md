# Daylight Blue v1.0.0

**Family:** Daylight

A lightweight standalone FrSky ETHOS theme.

- Focus: `invert`
- Controls: rounded
- Internal key: `DayBlu`
- Responsive 784x50 X20 / 464x50 X18 toolbar

Copy `theme-daylight-blue` into the transmitter `scripts` folder, restart, and select **Daylight Blue**.
