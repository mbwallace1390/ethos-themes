# Deep Space v1.0.0

**Collection:** Cosmic

A standalone FrSky ETHOS radio theme with custom toolbar artwork.

- Focus: `outline`
- Controls: square
- Internal key: `DSpace`
- Static 784x50 toolbar
- Does not modify Rotorflight or RF Suite Lua files

Copy `theme-deep-space` into the transmitter `scripts` folder, restart, then select **Deep Space** under **System > General > Theme**.
