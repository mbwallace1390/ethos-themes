# Daylight Orange v1.0.0

**Family:** Daylight

A lightweight standalone FrSky ETHOS theme.

- Focus: `invert`
- Controls: rounded
- Internal key: `DayOrg`
- Responsive 784x50 X20 / 464x50 X18 toolbar

Copy `theme-daylight-orange` into the transmitter `scripts` folder, restart, and select **Daylight Orange**.
