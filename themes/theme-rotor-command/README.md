# Rotor Command v1.0.0

**Collection:** Flight Systems

A standalone FrSky ETHOS radio theme with custom toolbar artwork.

- Focus: `outline`
- Controls: square
- Internal key: `RotorCm`
- Responsive 784x50 X20 / 464x50 X18 toolbar
- Does not modify Rotorflight or RF Suite Lua files

Copy `theme-rotor-command` into the transmitter `scripts` folder, restart, then select **Rotor Command** under **System > General > Theme**.
