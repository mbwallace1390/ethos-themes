# Ember Signal v1.0.0

**Family:** Two-Tone

A lightweight standalone FrSky ETHOS theme.

- Focus: `outline`
- Controls: square
- Internal key: `EmbSig`
- Responsive 784x50 X20 / 464x50 X18 toolbar

Copy `theme-ember-signal` into the transmitter `scripts` folder, restart, and select **Ember Signal**.
