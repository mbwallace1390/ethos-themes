# Desert Tactical v1.0.0

**Collection:** Tactical

A standalone FrSky ETHOS radio theme with custom toolbar artwork.

- Focus: `outline`
- Controls: square
- Internal key: `DesTac`
- Responsive 784x50 X20 / 464x50 X18 toolbar
- Does not modify Rotorflight or RF Suite Lua files

Copy `theme-desert-tactical` into the transmitter `scripts` folder, restart, then select **Desert Tactical** under **System > General > Theme**.
