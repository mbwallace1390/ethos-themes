# Violet Circuit v1.0.0

**Family:** Two-Tone

A lightweight standalone FrSky ETHOS theme.

- Focus: `outline`
- Controls: square
- Internal key: `VioCkt`
- Responsive 784x50 X20 / 464x50 X18 toolbar

Copy `theme-violet-circuit` into the transmitter `scripts` folder, restart, and select **Violet Circuit**.
