# Molten Verdigris v1.0.0

**Family:** Molten

A lightweight standalone FrSky ETHOS theme.

- Focus: `outline`
- Controls: square
- Internal key: `MltVerd`
- Automatically selects 784x50 artwork on 800px radios and 464x50 artwork on standard X18 radios

Copy `theme-molten-verdigris` into the transmitter `scripts` folder, restart, and select **Molten Verdigris**.
