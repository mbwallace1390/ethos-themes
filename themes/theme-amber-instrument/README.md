# Amber Instrument v1.0.0

**Family:** Retro Instrument

A lightweight standalone FrSky ETHOS theme.

- Focus: `outline`
- Controls: square
- Internal key: `Amber`
- Responsive 784x50 X20 / 464x50 X18 toolbar

Copy `theme-amber-instrument` into the transmitter `scripts` folder, restart, and select **Amber Instrument**.
