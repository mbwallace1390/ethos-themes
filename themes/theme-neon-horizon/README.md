# Neon Horizon v1.0.0

**Collection:** Cosmic

A standalone FrSky ETHOS radio theme with custom toolbar artwork.

- Focus: `invert`
- Controls: rounded
- Internal key: `NeoHor`
- Responsive 784x50 X20 / 464x50 X18 toolbar
- Does not modify Rotorflight or RF Suite Lua files

Copy `theme-neon-horizon` into the transmitter `scripts` folder, restart, then select **Neon Horizon** under **System > General > Theme**.
