# RF Suite Blue v1.0.3

Status: Stable

This is the first confirmed working release of RF Suite Blue.

## Confirmed result

- Theme appears in the ETHOS theme selector.
- Rotorflight RF Suite selected menu tiles change from yellow to blue.
- The blue highlight is easier to read and visually cleaner.

## Important fix

The internal `system.registerTheme()` key was shortened from `RFSuiteBlue` to `RFBlue`. FrSky supplied themes use short internal keys, and the corrected key allows ETHOS to register the theme.

The package intentionally excludes `main.luac` so ETHOS creates a new compiled copy from the corrected `main.lua`.
